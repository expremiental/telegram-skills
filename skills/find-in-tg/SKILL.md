# /find-in-tg — Поиск по Telegram

Поиск по чатам Telegram пользователя.

**Запрос:** $ARGUMENTS

Если $ARGUMENTS пусто — спроси, что искать.

## Шаг 1: Получи список чатов

Используй `list_chats` (limit: 30).

## Шаг 2: Поиск

Используй `search_messages` с запросом пользователя в 10 самых подходящих чатах (личные сообщения, группы по теме). Limit 5 на чат.

Если запрос широкий (например, «ремонт», «вакансия») — ищи в большем количестве чатов. Если конкретный (имя человека, название проекта) — в меньшем, но точнее.

## Шаг 3: Покажи результаты

Формат вывода, сгруппировано по чатам:

```
🔍 Поиск: «запрос»

[Название чата] — N результатов
  • [дата] отправитель: краткое содержание
  • [дата] отправитель: краткое содержание

[Название чата 2] — N результатов
  • ...

---
Найдено: X сообщений в Y чатах
```

## Правила

- Группируй результаты по чатам
- Для каждого сообщения — дата и отправитель
- Длинные сообщения сокращай до 1 строки
- Если ничего не найдено — скажи об этом и предложи альтернативные варианты поиска
- Весь вывод на русском
