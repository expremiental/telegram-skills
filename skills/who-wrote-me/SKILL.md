# /who-wrote-me — Кто писал

Проверь, кто отправлял сообщения пользователю за последние часы. Следуй шагам:

## Шаг 1: Получи список личных чатов

Используй `list_chats` с параметром `chat_type="user"` и `limit=100` — это вернёт только DM (1 на 1 переписки).

Пропусти: Saved Messages (свой ID — получи через `get_me`), Telegram (ID 777000).

## Шаг 3: Прочитай свежие сообщения

Для каждого подходящего чата используй `list_messages` с `from_date` = сегодняшняя дата (формат YYYY-MM-DD), limit 10.

Оставь только сообщения от других людей (не от самого пользователя). Для определения используй `get_me`.

## Шаг 4: Покажи результат

Формат вывода:

```
👤 Кто писал

[Имя] — [время]
  «краткое содержание сообщения»
  ⚡ ждёт ответа / 💬 информационно

[Имя 2] — [время]
  «...»

---
Написали: N человек
Ждут ответа: M
```

## Правила

- Сортировка по времени, новые сверху
- Сообщения, которые явно ждут ответа — пометить ⚡
- Информационные (ответ не нужен) — пометить 💬
- Содержание — одна строка максимум
- Весь вывод на русском
