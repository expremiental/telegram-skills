# /digest — Дайджест Telegram

Сделай дайджест последних сообщений из Telegram пользователя. Следуй шагам:

## Шаг 1: Получи список чатов

Используй `list_chats` (limit: 20) — получи последние диалоги.

## Шаг 2: Выбери активные чаты

Из списка выбери чаты с непрочитанными сообщениями или свежей активностью (сегодня). Максимум 7 чатов. Приоритет:
1. Личные сообщения
2. Небольшие группы
3. Каналы

**Пропускай чаты с >1000 непрочитанных** — это скорее всего архивные или замьюченные чаты.

## Шаг 3: Прочитай сообщения

Для каждого выбранного чата используй `get_messages` (page_size: 10) — получи последние сообщения.

## Шаг 4: Покажи дайджест

Формат вывода:

```
📬 Дайджест Telegram

[Название чата]
• краткое содержание (1-2 предложения)
• если есть вопрос к тебе — выделить

[Название чата 2]
• ...

---
Всего чатов с активностью: N
```

## Правила

- Кратко — одна строка на чат, максимум две
- Если кто-то задал вопрос или ждёт ответа — выделить это
- Пропускай сервисные сообщения (вход/выход из группы и т.д.)
- Если в чате только стикеры/реакции — пропусти его
- Весь вывод на русском
